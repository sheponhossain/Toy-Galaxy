## üéØ Purpose
The goal of ToyGalaxy is to provide a seamless marketplace experience where toy sellers and buyers can interact. It demonstrates core web development concepts like **Authentication**, **CRUD operations**, **Protected Routing**, and **Responsive Design**.

---

## ‚ú® Key Features
* **Secure Authentication:** Google Login and Email/Password authentication using Firebase.
* **Dynamic Navbar:** Responsive navigation that tracks user login state and displays profile details with a hover effect.
* **Interactive Toy Catalog:** A visual grid showing toys with "View Details" functionality.
* **Data Persistence:** Fetching and displaying data dynamically from JSON files and APIs.
* **Protected Routes:** Ensuring pages like *Add Toy* and *Profile* are only accessible to logged-in users.
* **SweetAlert Integration:** Stylish success and error notifications for user actions.
* **Family Feedback System:** A dedicated review section for parents and collectors to share their experiences.
* **Responsive Slider:** A high-performance banner slider built with SwiperJS for marketing promotions.
* **Modern Animations:** Smooth scroll and element animations for a premium feel.
